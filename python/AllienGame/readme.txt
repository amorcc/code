---------------------------------------
-- 安装pygame
---------------------------------------
1.确定pip支持的版本
    python
    import pip; print(pip.pep425tags.get_supported())
2.到https://www.lfd.uci.edu/~gohlke/pythonlibs/#pygame下载对应的pygame的.whl文件
3.python -m pip install --user pygame-1.9.3-cp36-cp36m-win_amd64.whl 命令安装pygame
